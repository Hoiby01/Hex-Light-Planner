# Hex-Light-Planner
This is a web app for creating hex light layouts and calculating the dimensions and part usage. 

Left click on an empty hex to place a hex, click on a filled hex to erase a hex. Right click to pan. Scroll to zoom.
Set part limits on the left panel.
Erase all and recenter buttons are on the top pannel.


Created with Claude Sonnet 2.4
